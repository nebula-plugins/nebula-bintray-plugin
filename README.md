nebula-bintray
==============

Addition Bintray tasks

## Usage

### Applying the Plugin

To include, add the following to your build.gradle

    buildscript {
      repositories { jcenter() }

      dependencies {
        classpath 'com.netflix.nebula:nebula-bintray:1.12.+'
      }
    }

    apply plugin: 'nebula-bintray'
